# JavaScript Slideshow WebApp

A responsive, interactive slideshow web application built with vanilla JavaScript, HTML, and CSS.

## Features

- **Auto-play** with customizable speed
- **Manual navigation** with arrow buttons and dots
- **Keyboard support** (Arrow keys and Spacebar)
- **Play/Pause** functionality
- **Speed control** (Faster/Slower buttons)
- **Responsive design** for mobile devices
- **Smooth fade animations**

## How to Use

1. Open `index.html` in a web browser
2. Use the arrow buttons or keyboard arrows to navigate
3. Click dots to jump to specific slides
4. Use Play/Pause to control auto-play
5. Adjust speed with Faster/Slower buttons
